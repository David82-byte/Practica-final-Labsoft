Práctica final - AngularJS + Node.js

Cómo ejecutar el proyecto

Para evaluar esta práctica correctamente y evitar bloqueos de seguridad del navegador con los archivos multimedia (CORS/YouTube), es obligatorio levantar el servidor Node.js, que actuará también como servidor de archivos estáticos.

Pasos a seguir:

1. Abre una terminal y navega hasta la carpeta del servidor:
   cd "minitwitter - server"

2. Instala las dependencias necesarias (si no están instaladas):
   npm install express sqlite3 cors jsonwebtoken

3. Arranca el motor del servidor:
   node server.js

4. Abre tu navegador web y entra exactamente en esta ruta
   http://localhost:3000/index.html
